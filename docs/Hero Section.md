# Eventra

Create Events. Book Tickets. Manage Everything.

Eventra is a full-stack event management and digital ticketing platform built with React, Django REST Framework, and PostgreSQL.

## Features

- User Authentication & Authorization
- Event Creation & Management
- Free and Paid Events
- Ticket Booking System
- Coupon & Discount Management
- Dummy Payment Simulation
- QR Code E-Tickets
- Email Ticket Delivery
- Ticket Check-In Scanner
- Organizer Dashboard
- Admin Management Panel
- Analytics & Reporting

## Tech Stack

### Frontend
- React
- TypeScript
- Vite
- TailwindCSS
- React Query
- React Router

### Backend
- Django
- Django REST Framework
- JWT Authentication

### Database
- PostgreSQL

### Other Services
- QR Code Generator
